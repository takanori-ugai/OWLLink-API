OWLlink Protégé Plugin

This plug-in makes any OWLlink reasoner available in Protégé

Requirements

  • Protégé 4.1
  • OWLlink-aware reasoner or
  • OWLlink server adapter which turns any OWL API reasoner into standalone OWLlink servers bundled with the OWLlink API available at http://owllink-owlapi.sourceforge.net

More information

More information about OWLlink can be found at www.owllink.org. Information about the OWLlink server adapter can be found at here.
